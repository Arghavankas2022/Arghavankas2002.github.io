# Minimal Cool Resume Website

## Features
- Minimal, modern design with gradient backgrounds
- Smooth animations and hover effects
- Responsive design (mobile-friendly)
- Subtle particle background
- Intersection Observer animations
- Easy to customize

## Quick Start
1. Open `index.html` in your browser to preview
2. Edit content in `index.html` 
3. Style tweaks in `styles.css`
4. Custom JS in `script.js`

## Deploy to GitHub Pages
1. Create a new GitHub repository named `yourusername.github.io`
2. Upload all files to the repository
3. Enable GitHub Pages in Settings > Pages
4. Site will be live at `https://yourusername.github.io`

## Customization
- Replace `Your Name`, job titles, dates, descriptions
- Update contact links and skills list
- Add your real projects with live links
- Modify colors in `styles.css` (search for gradient)
